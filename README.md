# Note App Project

You can save your notes by registering in this program. And refer to it whenever you need.

## Demo Project:
> **https://note-app12.netlify.app/**

## Package in Project:

- firebase
- react-icons
- react-router-dom
- tailwindcss
- compressorjs

## Feature

- Sign In
- Sign Up
- Forgot password
- Set photo for profile
- Edit profile
- Add note
- Edit and Delete note
